# 标准工作流
> 最后更新：2026-04-09

## 远端训练
**触发**：需要在 GPU 服务器上启动训练
1. 本地 git add + commit + push
2. rsync 同步到远端(含 configs/ + ugv_dqn/ + scripts/)
3. SSH: `conda run --cwd $PROJ -n ros2py310 python train.py --profile $PROFILE`
4. 检查日志确认启动成功
**注意**：必须 `--cwd <绝对路径>`,conda run 不会自动 cd

## 引用核查
**触发**：论文中需要添加引用
1. WebSearch / Semantic Scholar 定位
2. DOI 用 2+ 源确认
3. `curl -LH "Accept: application/x-bibtex" https://doi.org/<DOI>`
4. 确认 claim 在原文中存在
5. 失败标 `[CITATION NEEDED]`,严禁凭记忆生成 BibTeX

## 代码修改
**触发**：需要修改项目代码
1. 先 git push 备份
2. 输出 3-7 步计划 + 文件清单 + 风险 + 验证
3. 等用户确认后再动手
4. 修改完运行验证
5. 新增 `configs/repro_YYYYMMDD_<topic>.json`(纯文档改动豁免)

## 论文润色
**触发**：需要改写论文段落
1. 多 Agent 并行搜同领域真实句子
2. 提炼句式特征
3. 按句式改写并标注对标原句
4. 自检是否丢失信息

## 记忆检索(bigmemory)
**触发**：需要项目历史上下文
- Droid: 用 Task tool 派 memory-worker(gpt-5.4)检索 bigmemory/
- Claude Code: 用 `claude -p --model sonnet` 或直接 Read/Grep
- 热区优先,冷区按需 Grep
